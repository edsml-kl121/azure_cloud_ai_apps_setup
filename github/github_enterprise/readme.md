# GitHub Enterprise

Please head to: `https://github.com/account/enterprises/new` to create a new enterprise account.
This folder contains two file that focus on **GitHub Enterprise setup** but target **different use cases**:

| File Name | Description | Intended Audience |
|------------|--------------|-------------------|
| **Github EMU OIDC Tutorial.docx** | Provides a step-by-step guide on setting up **GitHub Enterprise Managed User (EMU)** with **OpenID Connect (OIDC)** integration. This tutorial focuses on enterprise-level identity federation and secure single sign-on (SSO) setup using **Microsoft Entra ID**. | System administrators or DevOps engineers managing enterprise GitHub organizations linked to Entra ID. |
| **Github enterprise personal with Azure subscription.docx** | Explains how to connect a **personal GitHub Enterprise account** to an **Azure subscription** for billing, deployment, or DevOps integration. It focuses more on the **Azure side of configuration** rather than identity setup. | Individual developers or small teams who want to use GitHub Enterprise with their personal Azure accounts. |


Note: Github enterprise personal with Azure subscription.docx is faster to set up than Github EMU OIDC Tutorial.docx.
